Что выведет программа? Объяснить вывод программы.

package main

import (
    "fmt"
)

func main() {
    a := [5]int{76, 77, 78, 79, 80}
    var b []int = a[1:4]
    fmt.Println(b)
}


Ответ:

Программа выведет:
[77 78 79]

Это происходит потому, что мы создаем срез b, который ссылается на часть массива a с индексами от 1 до 3.
